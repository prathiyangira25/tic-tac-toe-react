# Tic-Tac-Toe Game using React

This is a simple two-player Tic-Tac-Toe game developed using React.js.  
The application allows two users to enter their names and play the game interactively in the browser.

---

## Features

- Home page with game rules
- Player name input before starting the game
- Start button enabled only after entering both names
- Turn-based gameplay between two players
- Winner detection and draw detection
- Replay option after game completion
- Back button to return to the home page
- Simple and clean user interface

---

## Technologies Used

- React.js
- JavaScript (ES6)
- HTML
- CSS
- Node.js and npm

---

## How to Run the Project

1. Clone the repository:
   git clone <repository-url>
2. Navigate to the project folder:

    cd tic-tac-toe
3. Install dependencies:

npm install

4. Start the application:

npm start

5. Open the browser and go to:

http://localhost:3000

## Note

This project supports only two human players

No AI or single-player mode is included
